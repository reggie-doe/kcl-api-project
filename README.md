# KCL API Project

Small experimental API for the KCL research demo.  
This repository contains a prototype Flask app, some config, and sample data.

## Quick start
1. Copy `.env.example` → `.env` and fill values.
2. `pip install -r requirements.txt`
3. `python app.py`

> NOTE: This repo is actively developed — expect frequent small commits and temporary files in `backups/`.
